| 维度 | JY传感器 | KEYENCE传感器 | 胜出 |
|------|----------|---------------|------|
| X轴线性度 | 0.0104 %FS | **0.0038 %FS** | KEYENCE |

#### 📊 性能比值

| 指标 | KEYENCE相对JY的提升 |
|------|---------------------|
| X轴线性度 | 优 **2.74倍** (0.0104/0.0038) |

### 5.3 结论

1. **两款传感器均表现优异**：所有测试项线性度误差均 < 0.02 %FS

2. **KEYENCE整体表现更优**：
   - X轴线性度优于JY约 **2.7倍**
   - Z轴线性度优于JY约 **1.6倍**

3. **精度水平**：
   - JY传感器：X轴 ±4.2μm，Z轴 ±0.8μm
   - KEYENCE传感器：X轴 ±1.5μm，Z轴 ±0.5μm

4. **应用建议**：
   - 对于高精度测量需求，**推荐KEYENCE**
   - JY传感器性能也完全满足大多数工业测量要求

## 📐 七、计算公式

### 深度值转换
```
深度值(μm) = (原始16bit值 - 32768) × 0.4
```

### X轴位置转换
```
X位置(mm) = 像素位置 × 0.0125
```

### 线性度误差
```
线性度误差(%FS) = (最大残差 / 满量程) × 100%
```

### R²计算
```
R² = 1 - (残差平方和 / 总平方和)
   = 1 - Σ(y - ŷ)² / Σ(y - ȳ)²
```
